# 🍟 مطعم أبو النور - نظام المنيو الرقمي (Premium Dark Edition)

نظام متكامل لإدارة منيو المطعم الرقمي، مصمم بوضع ليلي (Dark Mode) فاخر ومريح للعين.

## 📁 هيكل المشروع
- `index.html`: الصفحة الرئيسية (المنيو) للزبائن.
- `cart.html`: صفحة سلة المشتريات.
- `admin-asasfasf-12412412-sdgsd.html`: لوحة التحكم السرية (تم تمويه الاسم للأمان).
- `styles.css`: التصميم العام للموقع.
- `script.js`: المنطق البرمجي.

## 🚀 كيفية الاستخدام
1. **المنيو**: تصفح، بحث، وإرسال الطلب عبر واتساب.
2. **لوحة التحكم (`admin-asasfasf-...html`)**:
   - كود الدخول الافتراضي: `1234`.
   - ارفع الصور وسيقوم النظام بضغطها ورفعها تلقائياً.
